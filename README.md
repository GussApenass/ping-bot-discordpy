# Ping Bot - Discord.py

Um bot simples, rápido e elegante utilizando os novos Components liberados pelo Discord! Ele possuí uma base robusta e **escalável** para quem quer começar um projeto em discord.py rapidinho! Ele possúi um comando de ping em slash command, e um comando de ping em prefixo.
Ambos respondem com o mesmo container.

Este template foi desenvolvido por GussApenass com muito amor e carinho!

---

## ✨ | Funcionalidades

- **Slash Command `/ping`**
- **Prefix Command `ping`**
- Responde automaticamente quando o bot é mencionado
- Estrutura robusta e escalável
- Interface moderna com **Components V2**
- Totalmente compatível com a **Shard Cloud**!

---

⚙️ **|** Caso queira rodar em sua máquina:

1️⃣ Clone o repositório em sua máquina:

  ```
  git clone https://github.com/GussApenass/ping-bot-discordpy
  cd ping-bot
  ```

2️⃣ Crie o arquivo `.env`:
Ele deve conter:
- **BOT_TOKEN** -> *O Token do seu BOT.* **OBS:** Nunca compartilhe o token do seu bot, usuários mal-intencionados podem usar este token para fazer maldades.
- **BOT_PREFIX** -> *O Prefixo do seu BOT*. Opcional, vem como padrão **!**.

3️⃣ Execute o bot e seja feliz!
